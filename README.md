# Sucata Cósmica S.A. 🛰️

> Jogo incremental/clicker de reciclagem espacial — colete sucata cósmica, automatize com drones e estações, e acione a singularidade!

## 🎮 Como Jogar

1. **Clique** no botão central para coletar sucata
2. **Compre** Drones Coletores e Estações de Reciclagem para automatizar a produção
3. Quando tiver sucata suficiente, **acione a Singularidade** para ganhar Matéria Escura
4. Matéria Escura dá um **bônus permanente** a toda a produção (+1% por unidade)
5. Repita e vá cada vez mais longe! 🚀

## ⚙️ Mecânicas

| Sistema | Descrição |
|---|---|
| Custo Exponencial | Cada unidade comprada encarece a próxima em ×1.15 |
| Produção Automática | Drones (+0.1/s) e Estações (+5.0/s) geram créditos |
| Prestígio | Reseta o progresso em troca de Matéria Escura permanente |
| Bônus Global | Matéria Escura multiplica toda a produção e o clique |

## 🚀 Deploy

O jogo é um único arquivo HTML estático. Para fazer deploy no GitHub Pages:

1. Faça push deste repositório para o GitHub
2. Vá em **Settings → Pages**
3. Em **Source**, selecione **GitHub Actions**
4. O deploy será automático a cada push na branch `main`

Ou acesse diretamente abrindo `index.html` no navegador.
